# Search Chat Interface

Interface de chat moderna para busca por similaridade com LLM.

## Instalação

```bash
cd frontend
npm install
```

## Desenvolvimento

```bash
npm run dev
```

A aplicação estará disponível em `http://localhost:3000`

## Build

```bash
npm run build
npm start
```

## Características

- Interface moderna 
- Layout responsivo com TailwindCSS
- Scroll automático de mensagens
- Integração com backend FastAPI
- Exibição de resultados em cards
- Score formatado em porcentagem

## Integração com Backend

O frontend conecta automaticamente ao backend em `http://127.0.0.1:8000/search`
